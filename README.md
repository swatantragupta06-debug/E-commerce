# 🚀 THE HARDOI STORE - Complete E-Commerce Website

A modern, fully responsive shopping website built using HTML, CSS, JavaScript and Firebase. Deployed on Vercel.

---

## 👥 Customer Features
- Firebase Authentication (Signup/Login)
- Product listing with image, name and price
- Place order (Cash on Delivery)
- Free delivery on first order
- ₹20 delivery charge after first order
- Order status (Pending / Delivered / Cancelled)
- Cancel order (only if Pending)
- Review and rating after delivery
- Mobile friendly design

---

## 👨‍💼 Admin Features
- Secure admin login
- Add new products
- Delete products
- View all orders
- Mark orders as Delivered
- Cancel any order
- Generate printable order slip

---

## 🎨 UI Features
- Modern clean design
- Hero section with banner
- Floating free delivery popup
- Product cards with hover effect
- Smooth animations
- Mobile responsive layout

---

## 🛠 Tech Stack

## 🔥 Firebase Setup
1. Go to Firebase Console
2. Create a project
3. Enable Authentication (Email/Password)
4. Enable Firestore Database
5. Copy Firebase config
6. Paste it in firebase.js

---

## 🚀 Deployment
1. Upload all files to GitHub
2. Go to Vercel
3. Import your GitHub repo
4. Click Deploy
